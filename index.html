Color-grab

An easy way to grab a color from everywhere on the screen.

Download this project as a .zip file Download this project as a tar.gz file

Welcome to Color-grab.

Sometimes you need a simple grabber to pick up a color on the screen, well now you just simple open color-grab and you got it.

Version 1.0 and 1.1

How to use

You just need to have zenity installed. Then in a console just hit:

 $ sh color-grab.sh

Install zenity on Ubuntu

$ sudo apt-get install zenity

OBS.: this script only works on zenity version 2.32 or greater.

Version 2.0

How to use

You need ruby1.8 libglade2-ruby1.8 installed. Then in a console just hit:

   $ ruby color-grab.rb

or python

  $ python color-grab.py

Install ruby | python on Ubuntu

 $ sudo apt-get install ruby1.8 libglade2-ruby1.8

or

 $ sudo apt-get intall python

Or simple install the deb version, and use menu Applications to access Color-grab.
